# Server Setup Automation Project

A beginner DevOps project built to practice Linux automation using Python and Bash.

## What this project does

- Runs a Bash script to perform basic server setup
- Runs a backup script
- Uses a simple command-line interface
- Logs the execution process

## Technologies

- Python
- Bash
- Linux (Ubuntu / WSL2)

## Usage

```bash
python main.py --setup
python main.py --backup
```

## What I learned

- Running Bash scripts from Python with `subprocess`
- Using `argparse` to build a simple CLI
- Organizing a small Python project
- Basic Linux automation concepts

## Current Status

This is one of my first DevOps learning projects. The goal is to improve my Python, Bash, and Linux skills by building small practical projects.
